Decision tree is a kind of machine learning method. Decision tree is a machine learning method. It has a tree structure in which each internal node represents a judgment on an attribute, each branch represents an output of the judgment, and finally each leaf node represents a classification result.
Decision tree is a very common classification method which is a kind of supervised learning. Supervised learning is giving a bunch of samples, each sample has a set of attributes and a classification result, i.e., the classification result is known, then learning these samples yields a decision tree that gives the correct classification for the new data.
